# Bug Report – Search with Empty Field

## Bug ID
BUG-PRO-01

## Title
System allows empty search and returns blank page

## Environment
- Web application
- Browser: Chrome

## Priority
High

## Severity
Medium

## Description
When performing a search without entering any value in the search field, the system does not display any validation message or default behavior, resulting in a blank page.

## Steps to Reproduce
1. Access the e-commerce home page
2. Click on the search field
3. Do not enter any characters
4. Click on the “Search” button

## Expected Result
The system should display a validation message informing that the search field is mandatory or present a defined default behavior (such as most searched or best-selling products), without breaking the user experience.

## Actual Result
The system returns a blank page with no feedback to the user.
